Práctica de Arreglos en JavaScript

¿Qué son los arreglos?

Los arreglos (también llamados Array, Vector, Matriz o Listas) se utilizan para almacenar múltiples valores en una sola variable.

const cars = ["Ford", "Volvo", "BMW"];
Nota: Al declarar con const no se puede reasignar el arreglo completo, pero sí se pueden modificar sus elementos.

Creación de arreglos

Se puede crear un arreglo vacío y luego proporcionar los elementos:

const cars = [];
cars[0] = "Saab";
cars[1] = "Volvo";
cars[2] = "BMW";

Acceso a elementos

Se puede acceder a los elementos por su índice (comenzando desde 0):

let cars = ["Ford", "Volvo", "BMW"];
let name = cars[0]; // "Ford"

Modificación de elementos

Se puede cambiar el valor de un elemento accediendo a su índice:

let cars = ["Ford", "Volvo", "BMW"];
cars[0] = "Toyota"; // Cambia el primer elemento

Mostrar el arreglo completo

Se puede acceder al arreglo completo mediante su nombre:

const cars = ["Saab", "Volvo", "BMW"];
document.getElementById("demo").innerHTML = cars;
