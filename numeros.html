Números en JavaScript

JavaScript tiene un solo tipo de número. Los números se pueden escribir con o sin decimales.

Ejemplos básicos

let x = 3.14; // Un número con decimales
let y = 3; // Un número sin decimales

Método toFixed()

Retorna un número con un número específico de decimales.

let x = 9.656;
x.toFixed(0); // "10"
x.toFixed(2); // "9.66"
x.toFixed(4); // "9.6560"
x.toFixed(6); // "9.656000"

Método Number()

Convierte variables de JavaScript en números.

Expresión Resultado
Number(true) 1
Number(false) 0
Number("10") 10
Number(" 10") 10
Number("10 ") 10
Number("10 años") NaN

Método parseInt()

Analiza un texto y devuelve el primer número encontrado.

parseInt("10"); // 10
parseInt("10.33"); // 10
parseInt("10 20 30"); // 10
parseInt("10 años"); // 10

Resumen de Métodos

JavaScript ofrece varios métodos para trabajar con números:

  • toFixed() - Formatea un número con decimales fijos
  • Number() - Convierte un valor a número
  • parseInt() - Analiza un string y devuelve un entero
  • parseFloat() - Analiza un string y devuelve un número con decimales
